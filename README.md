# Todo App

This is a project template using Vite, React, TypeScript, and Tailwind CSS v4.

## Features

- **Vite**: Fast build tool.
- **React**: UI library.
- **TypeScript**: Type safety.
- **Tailwind CSS v4**: Latest utility-first CSS framework using `@tailwindcss/vite`.
- **Interactive Setup**: A `postinstall` script prompts for the project name and package name after `npm install`.

## How to Use

1. **Clone the repository**:

   ```bash
   git clone https://github.com/ponkojdevbd/React-TypeScript-Tailwind-template.git my-new-project
   ```

2. **Install dependencies**:

   ```bash
   cd my-new-project
   npm install
   ```

3. **Setup**:
   The setup script will automatically run after installation and ask for:
   - **Project Name**: The display name of your project (updates `index.html` title).
   - **Package Name**: The name for `package.json`.

## Scripts

- `npm run dev`: Start development server.
- `npm run build`: Build for production.
- `npm run preview`: Preview production build.
- `npm run lint`: Run ESLint.
